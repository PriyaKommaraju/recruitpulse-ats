# RecruitPulse ATS

RecruitPulse ATS is a professional resume analyzer that simulates
Applicant Tracking System (ATS) screening used by recruiters.

## Features
- ATS score out of 100 (rule-based, deterministic)
- Keyword alignment, formatting & technical match
- Resume strengths and ATS-specific improvements
- Recommended job roles
- PDF resume upload
- Clean, professional UI

## Tech Stack
- Python (Flask)
- React (via CDN)
- Tailwind CSS
- Google Gemini API

## How to Run Locally
```bash
pip install -r requirements.txt
python main.py
